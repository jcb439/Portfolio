## Navigation Bar

The navigation includes:
- **"Julia Beitel"** (top left) - links to index.html
- **Three links**: Projects, About, Contact

## Footer

The footer has two columns aligned to the right:
- **Column 1**: Portfolio, About, Projects
- **Column 2**: Contact, LinkedIn (placeholder), Resume

## Getting Started

1. Extract this ZIP file to your repository
2. Open the folder in Visual Studio Code
3. Start editing the pages - they're currently empty but ready for content
4. Add your images to the `images/` folder
5. Update the LinkedIn URL in the footer (it's currently a placeholder)
